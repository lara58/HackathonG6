Facial Expression Recognition API

This API provides endpoints for facial emotion recognition.

Endpoints

  • GET /health - Check API status
  • POST /analyze - Analyze facial expression in an image
  • GET /analyze/mona-lisa - Get emotions for the Mona Lisa
  • GET /emotions - List supported emotions

Usage Example

fetch('/analyze', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        image: 'base64_encoded_image'
    })
})
.then(response => response.json())
.then(data => console.log(data));

Response Format

{
    "emotion": "happy",
    "confidence": 0.92,
    "all_emotions": {
        "angry": 0.01,
        "disgust": 0.02,
        "fear": 0.01,
        "happy": 0.92,
        "sad": 0.01,
        "surprise": 0.02,
        "neutral": 0.01
    },
    "faces_detected": 1,
    "face_location": {
        "x": 100,
        "y": 80,
        "width": 200,
        "height": 200
    }
}
