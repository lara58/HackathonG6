# API de Reconnaissance d'Expressions Faciales

Cette API permet de détecter en temps réel les expressions faciales (joie, tristesse, colère, surprise, dégoût, peur, neutre) via une webcam.

## Fonctionnalités

- Détection de visages en temps réel
- Reconnaissance de 7 émotions différentes
- Interface de test en temps réel
- Endpoint spécial pour analyser l'émotion de Mona Lisa
